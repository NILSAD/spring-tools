# spring-tools
Spring Boot/Cloud Tools

| Module              | Description                  |
|---------------------|------------------------------|
| `tool-dependencies` | Maven 依赖版本管理                 |
| `tool-common`       | 定义基础 pojo 类、枚举、工具类等          |
| `tool-framework`    | Java 框架拓展                    |
| `tool-api-server`   | 服务端入口程序                      |
| `tool-generator`    | 代码生成器：Mybatis-Plus Generator |
